# Petstore API Test Automation (CRUD)

This project contains automated API tests for **Swagger Petstore** using the **/pet** endpoints.

It covers:
- CRUD operations (Create, Read, Update, Delete)
- Positive and negative scenarios
- Screenshot capture on test failure
- Parametric browser execution (Chrome / Firefox)
- POM (Page Object Model) structure (API layer + UI base page)

Swagger Petstore API:
https://petstore.swagger.io/

---

## Tech Stack

- Python 3.x
- Pytest (test framework)
- Requests (API automation)
- Selenium (browser support for screenshot requirement)
- Webdriver Manager (automatic driver setup)
