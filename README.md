This study includes 3 automation test cases written in Python.
Case 1:
This case written in Python + Selenium. Test case could be able to run in Chrome and Firefox browsers.Coded according to POM requirements
Scope:
1-Visit the given page
2-Click to open positions and verified that listed successfully
3-Check that all positions listed by selected filter
4-Click to apply button and verified that job application form page displayed successfully
Case 2:
Code scenarios use Locust (1 user is enough).
Scope: Load tests and scenarios are needed to investigate the behavior of the search
module of given url, header and listing the results after the search.
Case 3:
This case contains CRUD operations API tests with positive and negative scenarios. Also POM (Page Object Model) structure used (API layer + UI base page).
